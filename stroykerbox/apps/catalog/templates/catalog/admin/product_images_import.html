Если в файле не используются заголовки для таблиц, тогда необходимо снять галочку с соотвествующей опции в форме выше.

Первая колонка-столбец должна содержать артикул нужного товара. Остальные - путь или url к импортируемому фото. Колонок с фото может быть произвольное кол-во.

Пути для картинок могут быть как путями относительно каталога с медиа, так и URL-ссылками на изображения, рассположенные на других сайтах.

Примеры ожидаемой структуры данных в файле для импорта

С заголовками:

любой заголоков для колонки с артикулом любой заголовок для фото 1 любой заголовок для фото 2 любой заголовок для фото ...
артикул-1 /путь/к/картинке/относительно/каталога/с/медиа/фото1.png http://какойтосайт.com/фото1.png ..
артикул-2 http://какойтосайт.com/фото2.png /путь/к/картинке/относительно/каталога/с/медиа/фото2.png ..

Без заголовков:

артикул-1 /путь/к/картинке/относительно/каталога/с/медиа/фото1.png http://какойтосайт.com/фото1.png ..
артикул-2 http://какойтосайт.com/фото2.png /путь/к/картинке/относительно/каталога/с/медиа/фото2.png ..

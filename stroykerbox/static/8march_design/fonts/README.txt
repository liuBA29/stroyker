Шрифт Blacker Sans Text (подписи под овальными картинками на главной 8 марта).

Чтобы подгружался Blacker Sans Text, положите сюда файлы:
  BlackerSansText-Regular.woff2   (приоритет)
  BlackerSansText-Regular.woff    (запасной вариант)

Имена файлов должны быть именно такими — их ожидает 8march_index_page_design.css.

Шрифт платный (Zetafonts / Blacker Sans Pro). Для коммерческого использования нужна лицензия.
Без файлов на сайте будет использоваться запасной шрифт Inter.

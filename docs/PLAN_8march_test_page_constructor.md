# План: тестовая страница 8 марта → конструктор из тегов

Цель: убрать хардкод с `8march_design_test.html`, вынести блоки в теги и собирать страницу через админку (контейнеры **new_design_middle** и **new_design_bottom**). Стили не трогаем — все классы и файлы CSS остаются как есть, только разметка переносится в шаблоны тегов.

---

## Именование для заказчика: new_design

**В тегах и во всём, что видит/трогает заказчик** (названия тегов в админке, подписи блоков) используем **new_design**, не 8march. Примеры: `render_new_design_test_tag`, `render_new_design_info_block`, подпись в списке — «new_design: тестовый блок». Внутренние классы CSS и пути к шаблонам можно не менять.

## Порядок выноса (по одному блоку)

1. **Блок «Гарантия качества / Доставка / Подарки»** (`index-info-8march`) — один тег, только разметка и текст, без данных из БД. Идеально для старта.
2. **Соцсети** (`index-social-8march`) — статика, один тег.
3. **Отзывы** (`index-reviews-8march`) — статика; позже можно подтянуть из API/БД.
4. **Карта и контакты** (`index-8march-questions-wrap`) — один тег, большой, но самодостаточный.
5. **Букет по желаниям** (`index-bouquet-wish-8march`) — тег с include формы.
6. **Сборные букеты** (`index-bouquets-8march`) — тег с данными из контекста (view).
7. **Герой** и **Коллекция 8 марта** — при желании позже (карусели, можно с конфигом/данными).

После каждого шага: тег добавлен в `helpers.NEW_DESIGN_TAG_LINES`, блок убран из хардкода страницы и появляется только если его добавили в контейнер в админке. Стили не трогаем — классы те же.

## С чего начинаем

**Первый блок: «Гарантия качества 2 дня / Бесплатная доставка / Большой выбор подарков» + рейтинги 2ГИС и Яндекс** (`index-info-8march`).

- Делаем тег **`render_new_design_info_block`**: шаблон с текущей разметкой и текстами.
- Регистрируем в списке тегов нового дизайна с подписью вида «new_design: гарантия качества / доставка / подарки».
- Убираем эту секцию из `8march_design_test.html`.
- В админке в «Новый дизайн: блоки главной страницы» добавляем элемент с этим тегом — блок снова появляется на странице, порядок можно менять.

Внешний вид и стили остаются как сейчас.

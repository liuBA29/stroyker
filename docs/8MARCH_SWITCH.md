# Переключение дизайна 8 марта

## Как выключить

В **своём** settings-файле (тот, который импортирует `base`), добавь одну строку:

```python
USE_8MARCH_HEADER_FOOTER = False
```

- Локально (dev): в `stroykerbox/settings/dev.py` добавь эту строку (или раскомментируй, если она уже есть в комментарии).
- У заказчика (прод): в файле настроек продакшена (например `settings/luciano.py` или как он у них называется) добавь ту же строку.

После изменения — перезапусти приложение. Сайт вернётся к старой шапке/подвалу и обычной главной.

## Как включить обратно

Удали строку `USE_8MARCH_HEADER_FOOTER = False` из своего settings (или закомментируй). Либо поставь:

```python
USE_8MARCH_HEADER_FOOTER = True
```

Перезапусти приложение — дизайн 8 марта снова включится.

---

По умолчанию дизайн 8 марта **включён** (в `settings/base.py` задано `USE_8MARCH_HEADER_FOOTER = True`). Переопределяешь только когда нужно выключить или включить у себя/у заказчика.

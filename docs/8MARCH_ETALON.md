# Дизайн 8march — эталон и инструкции

**Эталон:** проект `luciano-site-light` (соседняя папка или отдельный репозиторий).

При правках стилей и шрифтов сверяться с:

- **docs/8MARCH_DESIGN_TEMPLATES.md** — шаблоны, шрифты (§7), зафиксированные значения (§8: хэдер, герой, категории).
- **docs/8MARCH_FILES_FOR_DEPLOY.md** — список файлов для деплоя.
- **code/design/8MARCH_DESIGN_DEPLOY.md** — подключение стиля.

## Шрифты (как в эталоне)

Подключать в шаблоне страницы 8march:

- **Inter** (300, 400, 500, 600)
- **Cormorant Garamond** (300, 500, 700)
- **Geologica** (400)
- **Roboto** (300) — для заголовков секций (Roboto Light)

URL:  
`https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600&family=Cormorant+Garamond:wght@300;500;700&family=Geologica:wght@400&family=Roboto:wght@300&display=swap`

В CSS (`:root`) используются переменные:  
`--8march-font-heading`, `--8march-font-body`, `--8march-font-serif` (Cormorant Garamond), `--8march-font-cursive` (Geologica).

## Зафиксированные значения (§8)

- Кнопка «В КАТАЛОГ»: фон `#1a1a1a`, hover `#2d2d2d`, **без скруглений** (`border-radius: 0`).
- Стрелки героя: кружки `#e8c4c4`, hover `#dfb4b4`; иконки стрелок `#c49a9a`, hover `#b08a8a`.
- Подписи категорий: цвет `#c49a9a`, hover `#b08a8a`.
- Кнопки «СМОТРЕТЬ ВЕСЬ КАТАЛОГ»: фон `#FBE3E1`, текст `#B8736E`.

Не менять по своему усмотрению — только по запросу заказчика.

## Герой, вторая карточка — зафиксированная вёрстка

- **«БУКЕТЫ СПЕЦИАЛЬНО» / «ДЛЯ НЕЕ»** — перенос на новую строку перед «ДЛЯ НЕЕ» (`<br>` после «БУКЕТЫ СПЕЦИАЛЬНО»). **Не менять без явного указания заказчика.**
- **Слоган в заблюренном квадрате:** первая строка — «Скажите цветами то, что», вторая — «невозможно выразить словами» (`<br>` после «что»). Слоган прижат к низу квадрата (`margin-top: auto`). **Не менять без явного указания заказчика.**
